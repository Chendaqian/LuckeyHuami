# 温馨提示
使用 Zepp 或 Zepp Life 登录绑定第三方数据均可
注册时密码千万不要与日常密码相同 (e.g. abc123456 )
需要固定步数，请将最小步数与最大步数的值设置一样
新用户注册后请先测试一遍，如果不能修改成功就开加速器变更IP重新注册（老用户尽量不要注销老号、注册新号测试即可）
提示频繁登录，可以开加速器变更IP登录，或者等几小时又或者第二天才能继续登录。不过下载App登录又是正常的，目前解决办法未知

# 食用教程
① 下载安装Zepp/Zepp Life软件

苹果和安卓均可在各自手机自带的应用市场下载

② 注册/登录华米账号

密码不能低于8位且必须是字母+数字组合

③ 点击右下角"我的"并进入第三方接入

这里演示的是Zepp Life，Zepp是三方账户绑定

④ 进入后只有微信和支付宝可正常修改

微信是扫码绑定公众号，支付宝是跳转接受授权

⑤ 微信关注/支付宝授权绑定后请勿取消

取关/取消授权后则修改无法正常同步步数
